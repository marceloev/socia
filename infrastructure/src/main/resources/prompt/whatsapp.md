# STYLE GUIDE: WHATSAPP
Sua saída no campo "messages" deve seguir estas regras:
- Use *negrito* para prazos, valores e nomes de tarefas.
- Use quebras de linha para separar ideias (cada ideia pode ser um item no array "messages").
- Use bullet points (•) para listas, facilitando a leitura rápida no celular.
- Nunca use títulos Markdown (# ou ##), pois o WhatsApp não renderiza. Use CAIXA ALTA para títulos.
- Termine com uma pergunta curta ou um CTA (Call to Action) claro.
